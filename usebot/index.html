如何添加

# 方式一

通过 小念同学 (opens new window) 链接添加到频道

# 方式二

通过频道主页 -> 右上角 -> 机器人 -> 找到小念同学-> 添加到当前频道

# 方式三

通过其他频道 -> 点击小念同学头像 -> 添加到当前频道
