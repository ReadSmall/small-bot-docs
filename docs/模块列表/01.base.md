```

```

## 资料卡


## 排行榜

获取频道内的排行信息

### 连续打卡排行榜

获取连续打卡排名前五的用户

```
@小念同学 /排行榜 连续
```

### 累计打卡排行榜

获取累计打卡排名前五的用户

```
@小念同学 /排行榜 累计
```

### 积分排行榜

获取积分排名前五的用户

```
@小念同学 /排行榜 积分
```
