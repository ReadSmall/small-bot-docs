## 国庆头像

制作一张国庆头像

```
/国庆头像 <序号>
/国庆头像 <序号> @好友
```

::: tip
不带序号默认随机制作
<br>
既可以给自己制作, 还可以给好友制作哦
<br>
在指令后面@好友, 即可帮助好友完成制作
:::
