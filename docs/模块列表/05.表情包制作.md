::: tip
既可以给自己制作, 还可以给好友制作哦
<br>
在指令后面@好友, 即可帮助好友完成制作
:::

## 素描头像

制作一张素描头像

```
@小念同学 /素描头像 
@小念同学 /素描头像 @好友
```
