## 猜字谜

### 开始游戏

```
@小念同学 /猜字谜
```

### 回答

```
@小念同学 /猜字谜 答案
```

### 提示

```
@小念同学 /猜字谜 提示一下
```


### 结束游戏

```
@小念同学 /猜字谜 结束游戏
```

::: tip
猜字谜指令后面带上答案即可回答
<br>
例如: @小念同学 /猜字谜 晴空万里
:::
