## 绕口令

随机获取一句绕口令

```
@小念同学 /绕口令
```

## 计算器

算术题计算, 仅支持四则运算

```
@小念同学 1+1*33+32/21
```

## 查天气

查询当地的天气状况

```
@小念同学 深圳的天气怎么样
```

## 查战力 [英雄] [区服]

查询王者荣耀的英雄战力

```
@小念同学 /查战力 李白 安卓QQ
```

::: tip
区服说明
<br>
安卓QQ、安卓微信、苹果QQ、苹果微信
:::
