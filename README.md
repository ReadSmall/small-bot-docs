# 介绍

`小念同学`是一款完全公益性质的综合型QQ频道机器人。

`小念同学`所有服务完全免费，并不存在商业收费的现象，您可以将小念同学拉入自己的QQ频道享受其提供的服务，但请注意您在使用的过程中需要遵循其使用条款。
